# API Reference

## POST /api/encode
Загружает файл для кодирования в ДНК.

## POST /api/decode
Декодирует ДНК-последовательность обратно в файл.

## GET /api/status/{job_id}
Получение статуса задачи.

## GET /api/health
Проверка статуса сервиса.

## GET /api/report/{job_id}
Получение отчёта о кодировании.
